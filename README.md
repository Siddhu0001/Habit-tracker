# Habit Tracker

A clean, fast habit tracking app built with React and Vite. Add daily habits, mark them done, and watch your streaks grow.

## Features
- Add habits with categories and notes/reminders
- Daily streak tracking
- 7-day heatmap view per habit
- Edit and delete habits
- Dark mode toggle
- Today's progress overview
- Data persists locally (no sign-up, no backend)

## Tech Stack
- React 18
- Vite
- LocalStorage for data persistence

## Run locally
\`\`\`
npm install
npm run dev
\`\`\`
